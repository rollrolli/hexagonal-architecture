# 6. 도인 헥사곤 만들기 

**도메인 헥사곤**은 헥사고날 애플리케이션의 개발을 시작하는 곳 <br>
도메인을 기반으로 다른 모든 헥사곤을 도출한다.
핵심적인 기본 비즈니스 로직은 도메인 헥사곤에 있다.

## 값 객체 정의
- 엔티티는 식별자를 갖고 있는 시스템 컴포넌트를 분류하는 데 사용하는 요소
- 값 객체는 식별자를 갖지 않음
- 객체의 관련 엔티티와 값들을 캡슐화하는 애그리게잇
- 값 객체를 먼저 생성하는 것을 권장

## 도메인 서비스 정의
-  엔티티와 값객체를 사용하여 비지니스 로직을 처리하는 동작을 정의할 수 있다.
